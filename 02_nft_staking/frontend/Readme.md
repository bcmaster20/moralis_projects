## 1 Create react app && install Dependencies
 ```
 npx create-react-app nftapp
 npm i bootstrap@5.2.0-beta1
 npm i react-bootstrap
 npm i sf-font
 npm i axios
 npm i web3modal
 npm i @walletconnect/web3-provider
 npm i walletlink
 npm i web3
 npm i @alch/alchemy-web3
 ```
